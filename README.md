# EASE â€“ Exoskeleton for Assistive Support of Elderlies

EASE is a modular, lightweight, and energy-efficient lower-limb exoskeleton designed to support mobility in elderly individuals. This repository contains resources for the development, control, simulation, and power management systems of the EASE exoskeleton.

---

## Repository Structure
PowerSystems/ # Battery, buck converters, energy management
ControlSystems/ # Gait detection, motor control, safety features
Simulation/ # Gait cycle simulations and performance modeling
Docs/ # Literature review, diagrams, system design documents
SharedAssets/ # CAD files, visuals, datasheets

---

## Features

- Real-time energy management and fault detection
- Modular 3D-printed design for low cost and ease of maintenance
- Adaptive control strategies for various gait patterns
- Support for simulation and hardware-in-the-loop testing
